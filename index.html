OpenSwitchMaps helps you to switch between map services, such as Google maps, OpwnStreetMap, Bing map and so on. You can jump to another map service keeping the location and the zoom level

How to use

  1. First, add a bookmark(bookmarklet) to your browser. Drag following link to your bookmark bar: OpenSwitchMaps Web OR Bookmark following bookmarklet
    2. javascript:(function(){ window.location.href='https://tankaru.github.io/OpenSwitchMapsWeb/index.html#'+location.href;})();
  2. Open a map service
  3. Open saved bookmarklet
  4. You will move to a web page(here). And you will get links to jump to other map service.
